## Getting Started

Install the dependencies:

```bash
npm install
# or
yarn install
```

First, run the development server:

```bash
npm run dev
# or
yarn dev

# access: http://localhost:3000/
```

Run tests:

```bash
npm run test
# or
yarn test
```

You can build this application:

```bash
npm run build
# or
yarn build

# after this
npm run start
# or
yarn start

# access: http://localhost:3000/
```
